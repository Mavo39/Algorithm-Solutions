# Stack クラスを作成する問題


## 【Item クラス】
#### プロパティ
`int data: 要素の値`   
`Item next: 1つ先のノード`  
next はデフォルトでは null を設定  

## 【Stack クラス】
#### プロパティ
`Item head: 先頭のノード, デフォルトでは null に設定`  

#### メソッド
`void push(int data): スタックの先頭に Item を追加`  
`Integer pop(): スタックの先頭から Item を取り除き、その Item の値を返す。先頭が何も存在しない場合、null を返す。`
`Integer peek(): スタックの先頭の Item の値を返す`


## 【テストケース】
`s1 = new Stack()`  
`s1.push(2)`  
`s1.peek() --> 2`  
`s1.push(4)`  
`s1.push(3)`  
`s1.push(1)`  
`s1.pop() --> 1`  
`s1.peek() --> 3`  
  
`s2 = new Stack()`  
`s2.pop()`  
`s2.push(9)`  
`s2.push(8)`  
`s2.peek() --> 8`
