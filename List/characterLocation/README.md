# コマンドを表す大文字の文字列 commands が与えられるので、指示出しを与えられた後の位置を配列で返す、characterLocation という関数を定義する問題

## 条件
・キャラクターは原点（0,0）からスタートするとする。<br>
・コマンドの進行方向は、N: y 方向に +1、E: x 方向に +1、W: x 方向に -1、S: y 方向に -1 となる。<br>

## 関数の入出力例
入力のデータ型： `string commands`<br>
出力のデータ型： `integer[]`<br>

## テストケース
`characterLocation("NNNN") --> [0,4]`<br>
`characterLocation("NESW") --> [0,0]`<br>
`characterLocation("NW") --> [-1,1]`<br>
`characterLocation("AWFGMD") --> [-1,0]`<br>
`characterLocation("NSSNWENSSNNN") --> [0,2]`<br>

## 思考プロセス
・文字列を走査していく<br>
→ 文字がN、E、W、Sのいずれかに該当するときに指定されたを増減する<br>
→ x方向とy方向の管理が求められているため、xとyの数字を管理する変数を用意する<br>
→ 指定文字のときの数字が決まっているため、連想配列で管理する<br>

## 考えられるエッジケース
・文字列が空のケース<br>

## 悩んだ箇所
・特になかった。<br>

## 計算量
PHP<br>                                                                        
(予想)時間計算量：O(n)　空間計算量：O(1)<br>
・時間計算量：ループによる線形探索があるため<br>
・空間計算量：配列を生成するが、要素が2つに限定されており入力に依存しないため<br>
(結果)時間計算量：O(n)　空間計算量：O(1)<br>
