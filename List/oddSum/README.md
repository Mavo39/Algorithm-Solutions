# 整数で構成される配列内の奇数の合計値を計算する sumOfOddElement という関数を定義する問題

## 関数の入出力例
入力のデータ型： `integer[] arr`
出力のデータ型： `integer`

## テストケース
`sumOfOddElement([2,4,3,8,11]) --> 14`
`sumOfOddElement([1]) --> 1`
`sumOfOddElement([21,4,436,85,182,169]) --> 275`

## 解き方
・奇数かどうかを判定する関数を別に作成する<br>
・ループによって、配列内の数字を一つ一つ足していく<br>

## 感想
・シンプルな問題で非常に解きやすかった。<br>
