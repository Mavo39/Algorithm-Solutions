# Algorithm Solutions

このリポジトリでは、**RecursionCS で取り組んだアルゴリズム問題の解答・過程・気づきを記録** しています。

## 📌 目的
- **問題の理解**: 取り組んだ問題の詳細を残す
- **思考の過程**: 解法のアプローチと試行錯誤の記録
- **気づきの整理**: 学んだことや改善点を振り返る

## 📂 ディレクトリ構成
```plaintext
Algorithm-Solutions/
│── 問題名/
│    ├── README.md  # 問題の説明・解法・気づき
│    ├── python/    # Pythonの解答
│    ├── javascript/ # JavaScriptの解答
│    ├── php/       # PHPの解答
│    ├── java/      # Javaの解答
